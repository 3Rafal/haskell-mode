12 Compilation

Haskell Mode comes with an intelligent Major mode for compilation named HsCompilation defined in haskell-compile.el. This mode is derived from compilation-mode defined in compile.el.

First the compilation mode tries to find the project’s root directory. This is done by searching for files such as cabal.sandbox.config, stack.yaml, and something.cabal in that order.

If you have both cabal.sandbox.config and stack.yaml files defined, in your project’s root directory, cabal.sandbox.config takes precedence over stack.yaml and similary stack.yaml takes precedence over something.cabal.

You can start the compilation by M-x haskell-compile. The intelligent thing about this mode is that like it sets the project’s root directory. It will run the appropriate compilation command from the project’s root directory.
