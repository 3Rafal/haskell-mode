20 Shows documentation at point

haskell-doc-mode shows the Type of an identifier at point (if it exists). The type of functions/operators in Prelude are shown by default. Other information such as the structure of import, if, do etc. is also shown.

If you want this to work with other imported modules as well, then load the file into the repl. This can be done by pressing C-c C-l from the the current buffer, provided you have interactive-haskell minor mode enabled. Then when you place the point on a text for more than 0.5 seconds, the type information or other documentation information is shown.

To show type information for modules other than Prelude, we query the comint based GHCi process for type information.

20.2 Adding hook

Add your hooks to haskell-doc-mode-hook. By default, nothing is hooked to haskell-doc-mode-hook.
